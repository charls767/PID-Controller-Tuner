# Contributing to PID Controller Tuner

¡Gracias por tu interés en contribuir! Aquí están las pautas para hacerlo.

## Cómo Reportar Bugs

1. Usa el título descriptivo para el issue
2. Proporciona una descripción específica del bug
3. Incluye pasos para reproducir
4. Describe el comportamiento observado vs esperado
5. Si es posible, proporciona ejemplos específicos

## Proceso de Pull Request

1. Fork el proyecto
2. Crea una rama (`git checkout -b feature/AmazingFeature`)
3. Commit tus cambios (`git commit -m 'Add some AmazingFeature'`)
4. Push a la rama (`git push origin feature/AmazingFeature`)
5. Abre un Pull Request

## Estándares de Código

- Usa PEP 8 para Python
- Documenta funciones con docstrings
- Añade tests para nuevas features
- Mantén compatibilidad con Python 3.8+

## Reportar Vulnerabilidades

No abras un issue público. Contacta a los maintainers directamente.
